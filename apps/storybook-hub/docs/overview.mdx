# CDGPlayer Documentation Hub

This hub is the single source of truth for developer documentation.

What you can do here:

- Understand the package architecture
- Learn the contracts for loader/player/controls
- Copy implementation patterns for vanilla DOM and React
- Verify behavior through interactive composed stories

This hub composes both implementation styles:

- Framework-agnostic examples: `storybook-web`
- React examples: `storybook-react`
